Для начала работы, установите гит создайте папку проекта и репозиторий
Создайте и редактируйте файлы в папке, а затем отслеживайте изменения с помощью команды git add
Фиксируйте изменения с описанием используя git commit
Для совместной работы и хранения проекта в облаке создайте репозиторий на сайте и свяжите его с локальным репозиторием 
Загрузите изменения на гитхабе через git push 
Гит и гитхаб обеспечивают управление версиями, сотрудничество и историю изменений
